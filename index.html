A comprehensive logistics ecosystem engineered to streamline freight operations, inventory tracking, and client transparency for the Philippine market.

Overview

Dspeed Cargo PH was conceptualized to address the growing complexities in the domestic logistics sector. As a full-stack solution, it serves as the backbone for cargo handling, from the moment a package enters the warehouse to its final mile delivery.

The architecture focuses on high-availability and real-time data synchronization, ensuring that both administrative staff and customers have access to the exact status of their shipments without latency.

app/Services/ShipmentTracker.php 
public function updateStatus(Request $request) {
    $shipment = Shipment::findOrFail($request->id);
    $shipment->update([
        'status' => $request->status,
        'location' => $request->coords,
        'updated_at' => now()
    ]);

    event(new ShipmentMoved($shipment));
    return response()->json($shipment);
}
